#  Kindle3入手

[ Yuguo ](http://yuguo.us) 2010年 10月 09日

黑色的kindle3，尺寸刚好，很适合阅读。原生PDF是支持的，但是最好转成专有格式。

两点要注意的事情：

1.想读很多书，要注意按优先级排列阅读顺序。

2.要保持激情。

3.认真阅读，切忌浮夸。
