#  第二节羽毛球课

[ Yuguo ](http://yuguo.us) 2011年 04月 13日

发出快速的球的要诀不是更大的力量，也不是羽毛球拍的力矩，而是球拍碰到球的时候的速度，而这速度来自于运动距离。通过提前把球拍压在身后，击球时能更轻松地打出更强
力的球。

通过这样省力，能更加持久，也更加减少肌肉拉伤。

腰部的扭动跟小臂的爆发是最主要的速度来源。

重新学习动作规范很痛苦，因为全身上下都要有新的规则，常常记得这个忘了那个，不免受到教练苛责。

右臂肩膀很痛。
