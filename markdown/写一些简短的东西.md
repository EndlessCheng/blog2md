#  写一些简短的东西

[ Yuguo ](http://yuguo.us) 2011年 03月 11日

写简短东西的要诀：

  * 直入主题 
  * 语言准确而少 
  * 虽然小但对与此话题概括得较全面 
  * 比起140字的微博来说，可以有更多的元素，比如无序列表等富样式 
  * 比起长篇文章的好处：在首页就能读完，不必专门设置summary
