#  不患莫己知，求为可知也

[ Yuguo ](http://yuguo.us) 2011年 08月 04日

子曰：“不患无位，患所以立；不患莫己知，求为可知也。”

孔子说：“不怕没有职位，就怕自己没有学到赖以站得住脚的东西。不怕没有人知道自己，只求自己成为有真才实学值得为人们知道的人。”
