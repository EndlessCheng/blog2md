#  2.教程 

##  数据格式约定： 
    
    
    user:  你的手机号
    key：密码
    number：目的手机号
    text：要发送的信息
    

##  Get方式 

直接get地址 
    
    
    http://openfetionapi.sinaapp.com/fetion.php?&user=10086&key=10086&number=10010&text=hello,China Unicom
    

其中，user等参数替换为自己的信息 

##  Post 

目的地址： 
    
    
    http://openfetionapi.sinaapp.com/fetion.php
