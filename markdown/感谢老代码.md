#  感谢老代码

[ Yuguo ](http://yuguo.us) 2011年 01月 06日

感谢糟糕的老代码。

如果不是握有十全把握，我不会有勇气去批评和修改。为了具有说服力，我才会不断学习。

有时候并不是我超越了写糟糕代码的那个人，而是我所在的时代超越了他所在的时代，所以也没有什么值得骄傲的。

感谢优雅的老代码。

优雅的代码在任何时代都会像黑暗中的萤火虫一样光亮。

Keep coding.

update：写这篇文章是因为有时候看以前的代码真特么想骂娘——有时候这代码是自己写的。
